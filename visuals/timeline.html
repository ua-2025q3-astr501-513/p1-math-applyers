🌟 COMET-SEE Project Overview

Phase 1: Data Collection
📅 Week 1-2
Scraped comet catalogs from Sungrazer Project and downloaded SOHO/LASCO C3 FITS images for both comet observations and background sequences.
Years Covered: 2005-2021
Comet Sequences: 498
Background Sequences: 167
Total Images: ~25,000
Phase 2: Preprocessing
📅 Week 3
Applied difference imaging technique to highlight moving objects. Created frame-to-frame differences and maximum projections to reveal comet trails.
Difference Images: ~10,000
Max Projections: 665
Image Size: 512×512×3
Phase 3: Model Training
📅 Week 4-5
Trained EfficientNet-B0 model with data augmentation. Used 80/20 train/validation split with Adam optimizer and learning rate scheduling.
Architecture: EfficientNet-B0
Parameters: 5.3M
Training Epochs: 30
Best Val Accuracy: 97.7%
Phase 4: Evaluation
📅 Week 6
Comprehensive evaluation including confusion matrix analysis, precision/recall metrics, and comparison with baseline models.
Precision: 98%
Recall: 99%
F1-Score: 98.5%
Phase 5: Deployment
📅 Week 7
Deployed model as interactive web application on HuggingFace Spaces using Gradio. Created user-friendly interface for real-time comet detection.
Framework: Gradio
Platform: HuggingFace Spaces
Inference Time: ~2 sec/sequence
